== Description ==
This plugin contains Live Editors for 12 of the facebook official embed plugins and converts them to simple wordpress short codes

== Installation ==
1. Active Plugin
2. Set your application id and application secret within the plugin settings page
3. Change any optional settings inside {plugin-folder}/classes/config.php (for example html5 to html4)
4. Start generating shortcodes and enjoy this plugin :)
